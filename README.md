# Biosample_Library_System （生物样本库系统）
这里是一个在软件工程小学期内完成的生物样本库管理系统的基本样例 <br>
包括以下几个功能模块：<br>
    `样本入库`<br>
    `样本出库`<br>
    `库存管理`<br>
    `样本盘点`<br>
    `信息查询`<br>
    `样本采购`<br>
    `样本处置`<br>
具体执行流程图如下<br>
![image](https://github.com/Mr-strlen/Biosample_Library_System/blob/master/images/process_diagram.png)
  
系统使用使用基本SSM框架，eclipse+web组件作为开发ide，JRE 1.8，tomcat 8.5
